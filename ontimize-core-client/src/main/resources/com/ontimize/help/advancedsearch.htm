Búsqueda Avanzada

 

Para realizar búsquedas avanzadas puede utilizar los siguientes caracteres:

  • <, <=,=,>=,> al comienzo del texto. Por ejemplo: <12, o >=67
  • ':' para realizar búsquedas de datos correspondientes a un rango. Por ejemplo: 12:34 correspondería a 'entre 12 y 34' ambos incluidos
  • '|' para realizar busquedas de varios valores. Por ejemplo: 10/10/2002|01/10/2000|20/10/2001 corresponderiá a '10/10/2002 o 01/10/2000 o 20/10/2001'
  • '!' para realizar búsquedas con condición 'distinto'. Además, si solamente se introduce '!' en el campo la condición de búsqueda será que no haya datos para ese campo. Por ejemplo: '!24' corresponde a 'distinto de 24' y '!' corresponde a 'vacío'
