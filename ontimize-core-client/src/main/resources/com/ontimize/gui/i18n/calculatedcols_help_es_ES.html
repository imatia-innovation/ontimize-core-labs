Este componente permite  definir nuevas columnas calculadas. Para editar la expresión de  una  columna existente debe seleccionarse el nombre de  dicha  columna en la lista de la izquierda. Para insertar una  columna nueva utiliza el botón  Crear 
La expresión debe escribirse en el panel central,  para  hacer referencia a otra columna debe hacerse doble  click  sobre el nombre de la columna en la lista de la derecha.
Los distintos elementos tienen asociado un tip que puede resultar  de  ayuda.
