Este compoñente permite  definir novas columnas calculadas. Para editar a expresión dunha  columna existente debe seleccionarse o nome da  columna na  lista da esquerda. Para insertar unha  columna nova úsase o  botón  Crear 
A expresión debe escribirse no panel central, para  facer referencia a outra columna debe facerse doble click  sobre o nombre da columna na lista da dereita.
Os distintos elementos teñen asociado un tip que pode  resultar de axuda.
